# Contributing to QuizCraftQA-NotebookLM

Thank you for your interest in contributing!


## How to Contribute

- **Prompt Templates:**
  - Add new prompt templates to the `prompts/` folder.
  - Use clear instructions and formatting similar to existing files.
  - You can use `prompts/prompt_template_example.md` as a starting point.
- **Examples:**
  - Add new example outputs to the `examples/` folder.
  - Include a note about which prompt and syllabus section was used.
- **Documentation:**
  - Improve or clarify instructions in `README.md` or `CLAUDE.md`.

## How to Contribute New Prompts or Examples

1. Copy `prompts/prompt_template_example.md` and edit it for your new prompt type.
2. Save your new prompt in the `prompts/` folder with a descriptive name.
3. For examples, add a new file in `examples/` and mention which prompt and syllabus section was used.
4. Open a pull request with a clear description of your addition.

## Guidelines

- Keep prompts and examples focused on ISTQB syllabus content.
- Use clear, exam-style language.
- Reference syllabus sections where possible.
- Proofread for clarity and correctness.

## Submitting Changes

1. Fork the repository
2. Create a new branch for your changes
3. Make your edits
4. Open a pull request with a clear description

---

For questions, open an issue or contact the maintainer.
